# Time-forecasting-using-python
A time forecasting project using python (numpy, pandas, matplotlib and sklearn)
Problem Statement: Unicorn Investors wants to make an investment in a new form of transportation - JetRail. JetRail uses Jet propulsion technology to run rails and move people at a high speed! The investment would only make sense, if they can get more than 1 Million monthly users within the next 18 months. In order to help Unicorn Ventures in their decision, we need to forecast the traffic on JetRail for the next 7 months. 
Contents:
a) Understanding Data - 1. Hypothesis Generation
                        2. Getting system ready and loading the data
                        3. Data Structure and Content
                        4. Feature Extraction
                        5. Exploratory Analysis
b) Forecasting using Multiple Modeling Techniques - 
1. Splitting the data into training and validation part
2. Modelling techniques
3. Holt's Linear Trend Model on daily time series
4. Hold Winter's Model on daily time series
5. Introduction to ARIMA model
6. SARIMAX model on daily time series
